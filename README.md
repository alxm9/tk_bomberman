# Tkinter Bomberman

An implementation of Bomberman using python's Tkinter module.

## Local Game controls

### Player 1:

- **UP-DOWN-LEFT-RIGHT** - Move
- **SPACE** - Place Bomb

### Player 2:

- **WASD** - Move
- **F** - Place Bomb
